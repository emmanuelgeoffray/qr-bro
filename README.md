QR bro is a tool to generate QR codes in the [spacebro](https://github.com/spacebro/spacebro) galaxy.

## Install
```
sudo pip install qrcode
sudo pip install pathlib
sudo pip install socketIO_client
sudo pip install Wand
```

## Run

```
python qr-bro.py
```
